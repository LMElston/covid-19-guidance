Ce tableau énumère les comptes qui sont enregistrés pour le courrier en ligne et liés à l’adresse de courriel que vous avez sélectionnée. La première colonne comprend des cases à cocher pour que vous puissiez indiquer les comptes à séparer de l’adresse de courriel sélectionnée, et la deuxième colonne contient les genres de compte et les numéros de compte.
Il y a des options pour sélectionner tous les comptes ou tous les comptes dans un programme. Si vous faites une erreur, vous pouvez désélectionner des cases à cocher approprié afin d’effacer une sélection que vous avez effectuée. Vous pouvez effacer des sélections pour tous les comptes dans un seul programme, de même que pour tous les comptes dans tous les programmes. Si un seul compte est associé à l’adresse de courriel sélectionnée, il sera sélectionné d’avance pour vous.
