  • Pourquoi ma transaction n'a-t-elle pas été complétée?
    Les quatre raisons principales pour lesquelles une transaction n'est pas confirmée par votre institution financière :
    1. Vous avez entré un numéro de carte de crédit et Payez maintenant accepte seulement les cartes de Visa Débit, Débit MasterCard ou d'Interac en ligne.
    2. Vous avez peut-être excédé votre limite de retrait quotidien ou hebdomadaire établi par votre institution financière.
    3. Vous avez entré votre numéro de carte Interac dans le champ Visa Débit. Pour les transactions Interac en ligne, vous devez cliquer sur le bouton radio à côté du logo Interac et ensuite cliquer sur le bouton « Traiter la transaction ».
    4. Vous avez cliqué sur le bouton « Annuler la transaction » pour retourner à la page Web de l'ARC.
