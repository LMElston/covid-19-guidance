Form TF725

Form TF725, Registered Charity Basic Information sheet contains information CRA has on file for the charity. Verify that the pre-printed information on Form TF725 is correct, complete, and current. If the charity has misplaced this form, or if the details are blank or inaccurate, the charity can still complete this question by selecting Yes and using program areas field codes.

Identify the three primary areas in which the charity carried on programs during the fiscal period to achieve its charitable purposes. Rank the program areas according to the percentage of time and resources the charity devoted to each program area.
